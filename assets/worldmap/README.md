# World Map assets

v0.6.15.18 でワールドマップ用のPNG仮本番素材を追加しました。

## 使用中素材
- world_bg.png : ワールドマップ背景
- node_village.png : はじまりの村アイコン
- node_field.png : ひだまり草原アイコン
- node_cave.png : こけの洞窟アイコン
- node_forest.png : 精霊の森アイコン
- node_tower.png : 死の塔アイコン
- node_castle.png : 光の城アイコン
- node_ruins.png : 原始の谷アイコン
- player_pin.png : 現在地ピン

SVG版も同梱していますが、画面ではPNG版を参照しています。
今後、画像生成や手描き素材ができたら同名PNGで差し替えるだけで更新できます。
