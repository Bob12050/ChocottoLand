# ミニアランド v0.6.3 フォルダ整理版

v0.6.3「4方向歩行アニメ対応版」を、今後の大規模開発向けにフォルダ整理した版です。

## 起動方法

`index.html` をブラウザで開いてください。

GitHub Pages に上げる場合は、このフォルダ構成のままアップロードしてください。

## 主な構成

- `index.html`
- `css/style.css`
- `js/main.js`
- `assets/characters/`
  - プレイヤー素体の前・後・横・歩行A/B素材
- `assets/monsters/`
  - ぷるスライム素材
- `assets/ui/`
- `assets/field/`
- `assets/village/`
- `assets/equipment/`
- `data/`

## 実装内容

- 4方向向き切り替え
- 前向き / 後ろ向き / 横向き 歩行A/B
- 左向きは横向き素材を左右反転
- 移動中だけ歩行アニメ
- 停止時はAフレーム表示

## 注意

スマホで単体確認したい場合は、別途 `minialand-v0.6.3-walk-animation-smartphone.html` を使用してください。
このZIP版は、整理されたフォルダ構成で開発・GitHub公開するための版です。

### 武器の追加方法（メモ）
1. EQUIPMENT に武器を追加（slot:"weapon"）
2. EQUIP_RECIPES にレシピを追加（鍛冶屋に出る）
3. WEAPON_LOOKS に見た目を1行追加（type: sword/dagger/staff）

## v0.6.8 更新（武器を透過PNG画像で表示）

- 武器の見た目を「四角描画」から「透過PNG画像を手元に重ねる」方式に対応
  - WEAPON_LOOKS に img と向きごとの配置(view: down/up/side)を指定すると画像表示に切替
  - 配置データ：ox/oy（手元位置）, h（表示サイズ）, rot（傾き）, behind（後ろ向きで素体の裏に描画）, gripAnchor（画像内の握り位置）
  - 画像が無い武器は従来どおり四角描画（木の剣・鉄の剣・短剣・杖はそのまま）
- 炎の大剣をPNG画像表示に（assets/equipment/flame_greatsword.png）
  - 生成画像の市松模様背景を透過処理して使用
  - 前・後ろ・横の各向きで手元に重ねて表示、攻撃時は軽く突き出す演出
- 新しい絵武器の追加手順：①透過PNGをassets/equipmentに置く ②initSpriteAssetsでloadPngSprite ③WEAPON_LOOKSにimgとviewを指定

### 武器画像の作り方メモ
- 512x512・透過背景・「柄が左下／刃先が右上」の斜め1枚で生成
- 1枚を回転・反転で全向きに使い回す（向きごとに描き分け不要）
